# Brick Breaker Game

A classic arcade-style Brick Breaker game built using Python and the Turtle graphics library.

##  How to Play

1.  **Start the Game:** Run the script and press **ENTER** on the main menu.
2.  **Objective:** Use the paddle to bounce the ball and break all the blocks. Don't let the ball hit the bottom!

### Controls

| Key | Action |
| :--- | :--- |
| **A** | Move Paddle Left |
| **D** | Move Paddle Right |
| **Enter** | Start Game |

##  Installation & Setup

### Prerequisites
You need Python 3 installed. If you are on Linux, you also need the Tkinter library (which Turtle uses).

**Linux (Ubuntu/Debian):**

 - sudo apt-get update
 - sudo apt-get install 
 - python3-tk
 
## Running the Game
 - Open your terminal.
 - Navigate to the project directory:

```bash
 cd Break_breaker
```

## Run the game file:

```bash
 python3 game.py
```

 ## Features
 - Collision detection between ball, paddle, and blocks.

 - Randomized block colors.

 - Win/Loss mechanics.

 - Simple, lightweight graphics using Turtle.
